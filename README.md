## 📌 Main Features
### 1. Coredata
> + 다이어리 추가, 삭제
> + 다이어리 내 폴라로이드 이미지 추가, 삭제
> + 해시태그 검색
> + 최신순, 개수순 정렬
> + 드래그 앤 드랍으로 자유로운 폴라로이드 위치 변경
### 2. Edit pictures by adding stickers on polaroid frames
> + 스티커 추가, 삭제
> + 스티커 확대, 위치 변경
> + 폴라로이드 테두리 색상 변경
### 3. Memo
> + 애니메이션 : 편집 시엔 뒤집기, 저장한 후엔 스와이프로 카드플립
> + 폰트 변경
### 4. Save and share Image
